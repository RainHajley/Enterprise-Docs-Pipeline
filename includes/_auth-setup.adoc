=== Authentication and Credential Setup

To connect securely to the Vaultcore tenant gateway, you must authenticate using your enterprise service account keys.

. Run the configuration bootstrap command in your secure terminal:
+
[source,bash]
----
vaultcore auth login --tenant-id="enterprise-prod-01"
----

. Verify your active session token permissions:
+
[source,bash]
----
vaultcore auth status
----

> **Security Note:** Never store raw API secrets or private tokens in plain-text configuration files or public repositories.
